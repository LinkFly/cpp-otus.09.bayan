## Bayan - поиск одинаковых файлов

**Принцип работы**: программа спроектирована с учетом требований
минимальных обращений к диску. В соответствии с размером блока и алгоритмом хэширования - программа стремится за минимум 
обращений к диску вывести группы одинаковых файлов

```./bayan <размер_блока> <crc32|md5> <файл>*|<директория>```

**Параметры**
 - <размер_блока> - это размер блоков, на которые разбиваются файлы и которые подвергаются хэшированию (читаются одноразово)

 - <crc32|md5> - один из алгоритмов хэширования (может быть выбран только один)
 
 - <файл>*|<директория> - любое количество сравниваемых файлов
 
 - либо директория все файлы которой, надо сравнить (рекурсивно просматриваются все вложенные каталоги)

**Пример (запуск из директории ./build):**

```./Debug/bayan 5 crc32 "../tests/trivial_test/file1.txt" "../tests/trivial_test/file2.txt" "../tests/trivial_test/file3.txt"```
